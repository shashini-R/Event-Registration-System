About Our Event Registration System

The Event Registration System is a web application developed using HTML, CSS, Bootstrap, JavaScript, Node.js and Express.js.

This application helps users register for technical events such as Hackathons, Workshops, Seminars and Coding Contests.

The system allows users to:

  • Register for an event
  • View all registrations
  • Search registrations
  • Delete registrations

This project was developed as a Full Stack Web Development project for academic purposes.
